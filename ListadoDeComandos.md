# Manual de git

## Inicializar repositorio.

        git init

        git add README.md

        git commit -m "first commit"

        git branch -M master

        git remote add origin git@github.com:andrewsknight/python-files.git

        git push -u origin master


## Crear una rama. 

        git checkout -b develop

        git push

        git push --set-upstream origin develop
